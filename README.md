# 电动车充电安全实时智能监控报警系统

基于AI的电动车充电安全实时智能监控报警系统，通过集成AI视觉识别、电流监测及多传感器融合技术，实现对电动车充电过程的全面、精准监控。

## 主要功能

1. AI视觉识别
   - 异常行为检测
   - 火灾隐患识别
   - 违规操作监控

2. 电流监测
   - 实时电流数据采集
   - 异常电流检测
   - 电流趋势分析

3. 多传感器融合
   - 温度监测
   - 烟雾检测
   - 多源数据融合分析

4. 智能预警系统
   - 实时预警
   - 多级报警机制
   - 报警信息推送

## 技术栈

- Python 3.8+
- TensorFlow/PyTorch (深度学习框架)
- OpenCV (计算机视觉)
- FastAPI (后端API)
- React (前端界面)
- SQLite (数据存储)
- Arduino (传感器数据采集)
